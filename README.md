# About:
Miss all the pixel perfect art that was popular in the 80's? If so, Pixel Art Maker is for you! Try and replicate old classics like [Mario](https://en.wikipedia.org/wiki/Mario) or freestyle your own art! This playground app is here to promote creativity, but also bring the warm and fuzzy nostalgia associated with old videogames. ğŸ˜Š
